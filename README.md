# AI Retail Demand Forecasting & Inventory Intelligence Platform

## Project Overview

This project predicts Walmart weekly sales using Machine Learning and provides business insights through PostgreSQL, XGBoost, and Streamlit.

## Technologies Used

* Python
* PostgreSQL
* Pandas
* Scikit-Learn
* XGBoost
* Streamlit

## Features

* Data Cleaning
* Feature Engineering
* Sales Forecasting
* Random Forest Model
* XGBoost Model
* Interactive Streamlit Application

## Dataset

Walmart Weekly Sales Dataset

## Best Model

XGBoost

R² Score: 0.9779

## Project Workflow

Dataset → PostgreSQL → Feature Engineering → Machine Learning → Predictions → Streamlit Dashboard
